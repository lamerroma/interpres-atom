# Interpres-Atom

Локальний веб-сервер для перекладу тексту та документів через LLM.  
Розроблений для внутрішнього офісного використання без виходу в інтернет.

---

## Можливості

- Переклад тексту з SSE стрімінгом у реальному часі
- Переклад файлів зі збереженням форматування: **DOCX**, **PPTX**, **XLSX**, **PDF**, **TXT**
- 34 мови + автовизначення мови джерела
- Лічильник символів та перевірка лімітів перед відправкою
- Зупинка перекладу в будь-який момент
- Перегляд перекладеного DOCX прямо в браузері
- Лічильник онлайн-користувачів
- Адмін-панель: статистика, налаштування, конфігурація форматів

---

## Стек

| Компонент  | Деталі |
|------------|--------|
| Сервер     | FastAPI + Uvicorn |
| LLM бекенд | Ollama (OpenAI-сумісний API) |
| Модель     | `rinex20/translategemma3:12b` (за замовчуванням) |
| Порт       | `7860` |

---

## Структура

```
interpres-atom/
├── translate_server.py      # основний сервер (FastAPI, вбудований HTML)
├── docx_translate.py        # движок перекладу DOCX
├── pptx_translate.py        # движок перекладу PPTX
├── xlsx_translate.py        # движок перекладу XLSX
├── requirements.txt         # залежності
├── translator_config.json   # налаштування (генерується автоматично)
└── stats.db                 # статистика перекладів (генерується автоматично)
```

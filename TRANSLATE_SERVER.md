# translate_server.py — API документація

Локальний веб-сервер для перекладу тексту та файлів через Ollama.  
FastAPI + Uvicorn, інтерфейс — одна HTML-сторінка у браузері, відповідь стрімиться через SSE.

---

## Запуск

```bash
python translate_server.py
```

Інтерфейс: `http://<ip-сервера>:7860`

---

## Авторизація

Endpoints `/admin*` та `/config*` захищені HTTP Basic Auth.  
Логін і пароль зберігаються у `translator_config.json` (за замовчуванням: `admin` / `translate`).

---

## Конфігурація

Налаштування зберігаються у `translator_config.json` поруч зі скриптом.  
Файл створюється автоматично при першому запуску.  
Змінити можна через адмін-панель або вручну в JSON (потребує перезапуску).

| Параметр          | За замовчуванням                   | Опис |
|-------------------|------------------------------------|------|
| `base_url`        | `http://127.0.0.1:11434/v1`        | Адреса Ollama |
| `model`           | `rinex20/translategemma3:12b`      | Назва моделі |
| `max_tokens`      | `2048`                             | Максимум токенів у відповіді |
| `llm_timeout`     | `180`                              | Таймаут запиту до Ollama (секунди) |
| `chunk_size`      | `3000`                             | Розмір чанка (символів) при перекладі файлів |
| `temperature`     | `0.7`                              | Точність/творчість моделі (0–2) |
| `retry`           | `2`                                | Кількість повторів при невдалому чанку |
| `insert_mode`     | `replace`                          | Режим вставки в DOCX: replace / append / prepend |
| `separator`       | `\n`                               | Роздільник між оригіналом і перекладом (append/prepend) |
| `custom_prompt`   | `""`                               | Додаткова інструкція перед стандартним промптом |
| `max_pdf_pages`   | `10`                               | Ліміт сторінок PDF/DOCX |
| `max_chars`       | `30000`                            | Ліміт символів для тексту і TXT |
| `auth_user`       | `admin`                            | Логін адмін-панелі |
| `auth_pass`       | `translate`                        | Пароль адмін-панелі |

## Endpoints

### Публічні (без авторизації)

#### `GET /`
Повертає HTML веб-інтерфейс.

---

#### `GET /languages`
Повертає список підтримуваних мов (відсортовано за алфавітом, українські назви).

**Відповідь:**
```json
[
  {"label": "Англійська", "value": "English"},
  {"label": "Українська", "value": "Ukrainian"},
  ...
]
```

---

#### `GET /models`
Повертає список моделей, доступних на Ollama.

**Відповідь:**
```json
["rinex20/translategemma3:12b", "gemma3:12b", ...]
```

---

#### `GET /limits`
Повертає поточні ліміти для фронтенду.

**Відповідь:**
```json
{"max_chars": 30000, "max_pdf_pages": 10}
```

---

#### `POST /translate`
Переклад тексту. Відповідь — Server-Sent Events (SSE) стрім.

**Тіло (JSON):**

| Поле              | Тип    | Обов'язково | Опис |
|-------------------|--------|-------------|------|
| `text`            | string | так         | Текст для перекладу |
| `lang_from`       | string | так         | Мова оригіналу або `"auto"` |
| `lang_to`         | string | так         | Цільова мова |
| `system_msg`      | string | ні          | Перевизначає системне повідомлення |
| `prompt_template` | string | ні          | Перевизначає шаблон промпту |
| `context`         | string | ні          | Тип документу |

**SSE події:**

| `type`    | Поля               | Опис |
|-----------|--------------------|------|
| `id`      | `text`             | Унікальний ID запиту (для `/stop/{id}`) |
| `log`     | `text`             | Рядок лог-виводу |
| `token`   | `text`             | Черговий токен від LLM |
| `result`  | `text`             | Фінальний результат перекладу |
| `error`   | `text`             | Повідомлення про помилку |
| `done`    | —                  | Кінець стріму |

---

#### `POST /translate-file`
Переклад файлу зі збереженням формату. Відповідь — SSE стрім.

**Тіло (multipart/form-data):**

| Поле        | Тип  | Обов'язково | Опис |
|-------------|------|-------------|------|
| `file`      | file | так         | Файл (DOCX, PPTX, XLSX, PDF, TXT) |
| `lang_from` | str  | ні          | Мова оригіналу або `"auto"` (default) |
| `lang_to`   | str  | ні          | Цільова мова (default: Ukrainian) |

**Додаткові SSE події:**

| `type`      | Поля              | Опис |
|-------------|-------------------|------|
| `progress`  | `text`, `pct`     | Статус чанку та відсоток (0–100) |
| `download`  | `url`, `filename` | Шлях для завантаження та ім'я файлу |

---

#### `POST /translate-html`
Переклад HTML. Зберігає теги, перекладає тільки текстові вузли. Відповідь — SSE стрім.

Тіло і події аналогічні `/translate`.

---

#### `POST /stop/{request_id}`
Перериває активний переклад.

**Відповідь:** `{"ok": true}`

---

#### `GET /download/{file_id}`
Завантаження перекладеного файлу.  
`file_id` береться з події `download.url` після завершення `/translate-file`.

---

#### `GET /preview/{file_id}`
Перегляд перекладеного DOCX як HTML у браузері (конвертація через mammoth).

---

#### `POST /heartbeat`
Реєстрація активної сесії (пінг кожні 15 секунд).

**Тіло:** `{"session_id": "<uuid>"}`

---

#### `POST /heartbeat-leave`
Миттєве видалення сесії при закритті вкладки.

**Тіло:** `{"session_id": "<uuid>"}`

---

### Захищені Basic Auth (`/admin*`, `/config*`)

#### `GET /admin`
Повертає HTML адмін-панелі.

---

#### `GET /admin/stats`
Повертає статистику перекладів.

**Параметри:** `?limit=100` (кількість детальних записів)

**Відповідь:** `{"summary": {...}, "recent": [...]}`

---

#### `POST /admin/stats/clear`
Очищає всю статистику.

**Відповідь:** `{"ok": true}`

---

#### `GET /config`
Повертає поточну конфігурацію (JSON).

---

#### `GET /config/defaults`
Повертає стандартні значення конфігурації.

---

#### `POST /config`
Зберігає нову конфігурацію. Зміни набувають чинності одразу без перезапуску.

**Тіло (JSON):** об'єкт з будь-якими ключами з таблиці конфігурації.  
**Відповідь:** `{"ok": true}` або `{"ok": false, "error": "..."}`

## Шаблон промпту

За замовчуванням:

```
{text}

§ Translate the text above {direction}. This is a {context}.
Keep all ⟨P⟩ and ⟨N⟩ markers exactly in place — they mark paragraph and line breaks.
Preserve technical terms, proper nouns, and numbers exactly.
Do not add notes or commentary. Output ONLY the translation. §
```

**Плейсхолдери:**

| Плейсхолдер   | Значення |
|---------------|----------|
| `{text}`      | Текст для перекладу (з маркерами `⟨P⟩`/`⟨N⟩`) |
| `{direction}` | `"from de into uk"` або `"into uk"` (якщо Автовизначення) |
| `{lang}`      | Код цільової мови, напр. `uk` |
| `{context}`   | Тип документу, напр. `document` |

**Маркери `⟨P⟩` / `⟨N⟩`** замінюють `\n\n` і `\n` перед відправкою в LLM і відновлюються у відповіді.  
**`§...§`** — роздільник: сервер відкидає все після першого `§`, щоб прибрати коментарі моделі.
